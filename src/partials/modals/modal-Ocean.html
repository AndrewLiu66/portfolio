Ocean Data Lab

Visualize Hydrophone Data

My development work: view

Topic

Designed and developed a responsive website to help visualize and download hydrophone data for Ocean Data Lab led by Professor Shima Abadi

React front-end

The website is built on top of one of the most popular front-end frameworks React.js. UI components are created with MaterialUI, and map is powered by react-map-gl

Flask backend

Many scientific graphs can only be created with the help of Python libraries. Thus, I chose Flask, a micro web framework written in Python, to serve as the backend server for the website.

Modern Deployment

The interactive visualization tool is deployed on AWS EC2 for public usage
