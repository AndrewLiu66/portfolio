Full stack web development

Developed an online human resource website for recruiters to find employees easily.

React front-end

The website is built on top of one of the most popular front-end frameworks React.js. Components are created with MaterialUI.

Mock backend

Currently, we are still working on creating the backend server to serve the need of send data to the front-end website. Thus, All the data shown on the website are now mocked by npm package axios-mock-adapter.

Modern Deployment

The front-end website is currently hosted on Netlify for viewing purpose. When the backend is ready, we will deploy on Amazon AWS EC2.
